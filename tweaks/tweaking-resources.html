Tweaking Resources

Create your very own tweaks! —
Here I share the base files needed to create your very own tweaks!

If you encounter any problem with any of the files provided here, please contact me at Discord @luisarlz
These files are base files, meaning they should technically work for any server after editing. However most files inside the scripts folder will not be compatible outside its own server, so those must be manually tailored and used inside its own server. Other folders are mostly safe to use and edit.

Textures Folder
( Visual Effects )

Base folder to freely edit visual effects. All textures are already decrypted. Copy and Paste your edited files in their correct folders in any server after editing.
