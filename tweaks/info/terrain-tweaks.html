Floor textures tweak "updates" all your floor textures to a different visuals version. The edits of floor textures are meant to create more visually appealing textures, be it as darker colors or simply better, more modern visuals.
Terrain mod is a tweak I created because I wanted to look at a more beautiful game, even if a little, while doing my common game tasks. As usual, I had the idea while doing my common tasks in the toilet.

I based myself off the original 'Night Argent Tweak' as it's a very popular tweak... but I think it happens to not be... very visually nice. Still, this is just an alternative and not meant to replace the original Night Argent Tweak, it's simply something I consider a little improvement.

Back then, I created the Night Argent Tweak due to my eye problems, but at this time I had absolutely no idea how to change colors, so I just made every texture blacker... that's it. However, later on I created an HD mod, I made a careful edit keeping colors properly and whatnot, to turn it into an actual "Night" view (not just blacker colors). I used about 10 minutes of my precious time manually editing the colors, it's been hard. I named this version 'Night Version #1'. But well... since people seem to enjoy simplicity, I also made another version, which, like it's classic version, just makes every texture blacker, and called it 'Night Version #2'. 1 minute of hard work was spent creating this one.
  Folders to backup before installing: texture/terrain and texture/water 

  • Download the Creative Terrain Folder
  • Download Image Editing Tools

Go to your client's folder, texture > terrain. You will find a lot of images there which can be edited with any designing tool. I highly recommend using something like photoshop, gimp or similar, as integrated tools like paint might not give you the best results. You can check out This Guide to see more about such editing tools. Remember it is important to keep the image size and format intact, as modifying the image size or format might break the game. Also, as you may notice by now, the floor works in 'Tiles' (little squares that repeat), so whenever you edit it, make sure to make every tile seamless (this meaning every tile has to merge with each other in harmony) otherwise you will get a squary, patchy mess. You can always find tutorials on how to do this by simply looking up 'how to make seamless tiles in photoshop' and similar searches. You can always find seamless images in google as well which will give you a good helping hand!

Creative Terrain Folder Instructions:


  • Download the Creative Terrain Folder
  • terrain folder contains all the base game files
  • terrain2 folder contains a guide on where each file is used
  • put all terrain2 files in your texture/terrain folder. replace all and open the game
  • you should now see the name of each file printed on the ground
  • you may now manually edit anything you’d like, using the terrain folder or using any of the already pre-loaded tweaks in the Downloads Section Below.
  • In the case you see only completely white textures in your floor, it means that the tweak was not correctly installed. Please make sure you're not taking folders or images out of the folders and also make sure you're not changing the image's format and size.

  • If you don't see changes in your tweak after installation, make sure you’re dropping the files in their correct folders and properly replacing them. If you’re using a full tweak from my webpage, make sure you’re dropping the main folder in your main game’s folder. Do not take anything out.
