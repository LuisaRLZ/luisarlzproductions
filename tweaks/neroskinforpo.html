About Nero Skin for Pirates Online

Download Nero Skin for PO

  Folders to backup before installing: The whole scripts folder 

I sincerely do not entirely love Pirates Online's default skin. There is nothing wrong with it I simply do not love how much space each thing consumes, specially buttons and huds, so I made this tweak which simplifies most things to a minimal space usage.

Named it Nero as a reference to the old 'Black Skin' which nowadays just sounds wrong.

So far I have edited (Version 0.2):
- Made Player hub a lot smaller
- Made Pet hub smaller, though Growth hint doesnt work as it should yet, to be fixed
- Made mini map smaller, simplified area name, coordinates and server time
- Made right side buttons simple: Mall, Battle Pass, Database and VIP stuff
Also one of the oldest tweaks in this game. Back in 2008 we became familiar with Noir's Black Skin Tweak as what it did was to change your User Interface (UI) to a dark version. Back then that was the cool stuff! Then by 2011 it became necessary to avoid looking at ToP2's GIGANTIC UI assets. Since screens were not as big before, the UI alone covered almost a whole half of your game. Several versions of 'Black Skin' were released from 2011 to 2016 adding in new assets or better modifications. Maybe I am wrong, but I feel PO suffers from a similar issue where a couple assets cover too much screen space. As I said, there's technically nothing wrong with that, it is just a matter of a personal preference, hence I made this fast tweak.
Soon I will try releasing a video explaining everything in detail. However as a fast explanation - You gotta create the image you want to use for the background first, then the images for HP, SP, etc. Whatever you wish to add. Then... basically experiment with it.

As a general tip, you will be using the following files from scripts/lua/forms: main.clu, equip.clu, player.clu and minimap.clu
Once you see all the weird text, lines that begin with frmDetail = UI_CreateForm are the ones that will create the invisible background to put images onto, and the lines that begin with UI_LoadImage will be the ones you use to add your image. In both cases, the 4 numbers that go like 100, 15, 45, 2 etc mean this, in this exact order: Tall of image, Length of image, Placement of image from left to right, placement of image from up to down.
There might be some bugs I did not notice while on the making of this skin. If you happen to encounter any bug with it, please report it to me at Discord @LuisaRLZ#3429 . However if your game simply does not open, it means you either installed it into the wrong server (This mod is meant exclusively for Pirates Online!) or some file is just not working properly. Please make sure to backup your scripts folder before installing, if anything goes wrong, just put the backed up folder back into PO's folder, or simply run version check in your PO's launcher. If you install and see things changed BUT the UI is invisible, make sure you also put the folder named nero inside your main PO folder, as this folder is the one with all images information.
